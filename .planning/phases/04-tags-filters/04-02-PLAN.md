| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-04-08 | Tampering | Client-side AND filter | accept | Filter runs on already-loaded data; server RLS already restricts which attendees/tags are visible; no additional server exposure |
| T-04-09 | Elevation of Privilege | /me/tags page delete/rename | mitigate | PATCH/DELETE calls go to 04-01 routes which enforce created_by ownership check server-side; UI guard is UX-only |
